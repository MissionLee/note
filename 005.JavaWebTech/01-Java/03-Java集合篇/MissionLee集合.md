# 集合

### 我对集合的思考

我们为什么需要集合，需要什么样的集合。

- 集合类型
  - Set 集 ： 把需要的内容放在一起
    - HashSet
    - TreeSet
    - LinkedHashSet
  - List 列表 ：按照顺序放一些内容
    - ArrayList
    - LinkedList
    - Vector 矢量队列
    - Stack 栈
  - Map 映射 ： 键值对
    - HashMap
    - LinkedHashMap
    - TreeMap
    - WeakHashMap
    - IdentifyHashMap
    - HashTable
- 一些集合的特点(结构)
  - Hash
    - 基于散列
  - Tree
    - 树结构，在这种结构下，里面的元素会由一定的顺序
  - Linked
    - 链表结构
  - Array
    - 数组结构
